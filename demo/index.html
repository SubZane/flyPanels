flyPanels - an off canvas menu panel system for responsive websites

FlyPanels is a responsive off canvas panel plugin for websites or web apps. It supports all modern browsers and IE from version 10. It can be used for menus, contact forms or anything you like really.

Compared to many other off canvas menu plugins out there this one is more solid and behaves more like a native solution. Try it!
